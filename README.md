# Dead Frontier Script Collection

A set of user-scripts to enhance your Dead Frontier gaming experience with organized layout and quality-of-life improvements.

## üêæ Step by Step

1. Install [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) from Chrome Store.
2. Click the **Install** button for each script below (or visit the file and click raw).
3. After redirecting, click **Install** in Tampermonkey.
4. Refresh your Dead Frontier page.
5. If prompted, select **Always allow this domain**.

<a href="https://host.xeidamoka.com/DeadFrontier/Scripts/Layouts/File/Layouts.user.js"

### Features
- Comfortaa Font (change code to any Google Font)
- Organizes the side bar layout
- Adds icons to Outpost locations
- Hold icons to add to favorites (includes collection box)
- Dynamic weapon sidebar layout depending on weapons equipped
- Adds colored XP % for next level
- Removes % from stats and adds missing value
- Removes 100% text and adds green glow to the image
- Boss Map toggle on top of the page
- OA warning system in page
- Remove Facebook button
