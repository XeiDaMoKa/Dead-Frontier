# Dead Frontier Script Collection

A set of user-scripts to enhance your Dead Frontier gaming experience with organized layout and quality-of-life improvements.


## üêæ Step by Step

1. Install [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) from Chrome Store.
2. Click the **Install** button for each script below (or visit the file and click raw).
3. After redirecting, click **Install** in Tampermonkey.
4. Refresh your Dead Frontier page.
5. If prompted, select **Always allow this domain**.


Any kind of suggestions, problems or bugs let me know in Dead Frontier or github issues.

Scripts auto-update via GitHub. Enable them in Tampermonkey dashboard.


##
## Layouts Script
### Features
- Comfortaa Font (change code to any google font)
- Organizes the side bar layout
- Adds Icons to Outpost locations
- Hold icons to add to favorites (includes collection box)
- Dynamic weapon sidebar layout depending on weapons equipped
- Adds Colored XP % for next level
- Removes % from states and adds missing value
- Removes 100% text and add green glow to the image
- Boss Map toggle on top of the page
- OA warning system in page
- Remove Facebook button

![Layouts Script](https://i.ibb.co/xq6qhxss/vivaldi-QOM69-F0e-FC.png)

Install [Layouts](https://github.com/XeiDaMoKa/Dead-Frontier/raw/Dead%20Frontier%20-%20Layouts.user.js) Script

## Prices Script
### Features
- Lowest market prices on the tooltip popup on item hover (60sec) (Color Dependent)
- Displays the difference between scrap and market price
- For bullets it shows the full price for your quantity in the cheapest market per unit
- Collection Book in the tooltip (Simply visit each tab in the collection for each item type)

![Prices Script](https://i.ibb.co/xqg6ZGR2/vivaldi-px2-OEKJR7-F.png)
![Prices Script](https://i.ibb.co/S4YX0DYR/vivaldi-o8ifqg-PKIH.png)

## Market Script
### Features
- Colors the items in the market
- Drag and Drop of items in the Market Display to normal search
- Drag and Drop in the Search element to search for Service
- Drag and Drop in the Market icon from any page if you installed Layout script
- Displays only the cheapest Service per level
-Displays the value of the items sold in the messages
